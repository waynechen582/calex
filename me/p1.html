Python 爬蟲應用

本專案展示了如何使用 Python 實現網頁爬蟲技術。我們選用 Yahoo 電影清單作為爬取目標，透過 Python 的 `requests` 和 `BeautifulSoup` 套件，抓取特定電影資料，並將其匯出至 CSV 試算表內。該專案不僅鍛煉了我們的程式能力，還加深了對爬蟲應用的理解。

專案摘要

專案由三人合作完成，經歷 20 天的設計與開發，解決了多個技術問題並最終完成。專案亮點在於資料抓取的準確性和試算表匯出的客製化功能。

心得反思

這次專案讓我們學會了如何應對程式開發中的挑戰。從初步構想到實際實現，我們經歷了無數次的問題與挫折，但也從中獲得了解決問題的能力與成就感。Python 的爬蟲技術給了我們探索更多應用可能的動力，未來我們將繼續在程式設計領域中探索與進步。
